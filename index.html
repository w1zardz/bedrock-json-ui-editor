About Bedrock JSON UI Editor

A free, open-source visual editor for Minecraft Bedrock Edition JSON UI files. Perfect for editing scoreboard, HUD, shimmer effects, and any UI element positions without manually tweaking JSON values.

Features

  • Visual editing of offset, size, anchor, alpha, and layer properties
  • Element tree navigation with search
  • Live preview of UI element positions
  • Mobile-friendly interface for editing on any device
  • Works with PocketMine-MP (PMMP) resource packs
  • Supports all Bedrock JSON UI files: scoreboard, hud_screen, inventory, and more

How to Use

  1. Copy your JSON UI file content (e.g., scoreboard.json from your resource pack)
  2. Paste it into the editor
  3. Click "Parse & Edit" to load the element tree
  4. Select any element to adjust its properties visually
  5. Export the modified JSON back to your resource pack

Compatible With

Works with all Minecraft Bedrock Edition platforms: Windows 10/11, Android, iOS, Xbox, PlayStation, Nintendo Switch. Also compatible with PocketMine-MP, Nukkit, and other Bedrock servers.
